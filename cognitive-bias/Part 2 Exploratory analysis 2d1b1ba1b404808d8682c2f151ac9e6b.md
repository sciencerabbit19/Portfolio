# Part 2. Exploratory analysis

In this section, we conduct an exploratory data analysis (EDA) on the labeled dataset. The objective is to examine distributional patterns, identify potential associations between variables, and highlight structural characteristics of the data.

We begin by importing the required libraries and loading the saved dataset:

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# Load the labeled dataset generated in Part 1
df = pd.read_csv("../cognitive_bias/labeled_tweets.csv")
```

To gain an initial overview of the data, we visualize the distribution of truth labels across platforms, allowing for a preliminary comparison between BlueSky and Mastodon.

```python
# Create a figure with two subplots
fig, ax = plt.subplots(1, 2, figsize=(10, 6))

# Distribution of truth labels by platform (percentage-based)
sns.countplot(
    data=df,
    x="truth_label",
    hue="platform",
    stat="percent",
    ax=ax[0]
)

# Overall platform distribution (percentage-based)
sns.countplot(
    data=df,
    x="platform",
    hue="platform",
    stat="percent",
    ax=ax[1]
)

# Extract legend handles and labels from the first subplot
handles, labels = ax[0].get_legend_handles_labels()

# Create a shared legend for the entire figure
fig.legend(
    handles,
    labels,
    title="Platform",
    loc="upper center",
    ncol=len(labels)
)

# Remove duplicate legend from individual subplot
ax[0].legend_.remove()

# Adjust layout to accommodate the shared legend
plt.tight_layout(rect=[0, 0, 1, 0.9])
plt.show()
```

![image.png](Part%202%20Exploratory%20analysis/image.png)

The visualization suggests that a larger proportion of content labeled as **FALSE** originates from BlueSky compared to Mastodon, despite the smaller number of posts collected from Mastodon.

To formally assess whether this observed difference reflects a statistically significant association between platform and truth label, we perform a $\chi^2$-test of independence.

```python
from scipy.stats import chi2_contingency

# Create contingency table between truth label and platform
table = pd.crosstab(df['truth_label'], df['platform'])

# Perform Chi-square test of independence
chi2, p, dof, expected = chi2_contingency(table)

# Display p-value
print("p-value:", p)
```

```python
p-value: 7.313527170002524e-11
```

Since the p-value is well below conventional significance thresholds, we reject the null hypothesis of independence and conclude that there is a statistically significant association between platform and truth label.

In particular, posts from BlueSky exhibit a higher proportion of content classified as FALSE compared to Mastodon, according to the labels generated by the Phi-3 Mini model. It is important to note that this conclusion reflects the model’s labeling and not an independently verified ground truth.

A similar approach can be applied to investigate whether the distribution of truth labels varies across topics:

```python
# Distribution of truth labels across topics (percentage-based)
sns.countplot(data=df, x='truth_label', hue='topic', stat='percent')
```

![image.png](Part%202%20Exploratory%20analysis/image%201.png)

The visualization indicates that the anti-vaccination topic exhibits the highest proportion of content labeled as FALSE, whereas climate change shows the lowest proportion.

One possible interpretation is that vaccination-related discourse—particularly during the COVID-19 pandemic—has been strongly associated with misinformation and conspiracy narratives. In contrast, climate change discussions, while often polarized, do not appear to generate the same relative concentration of content classified as false within this dataset.

To determine whether the observed differences across topics are statistically significant, we apply the same $\chi^2$-test of independence used in the previous analysis.

```python
# Create contingency table between truth label and topic
table = pd.crosstab(df['truth_label'], df['topic'])

# Perform Chi-square test of independence
chi2, p, dof, expected = chi2_contingency(table)

# Display p-value
print("p-value:", p)

```

```python
p-value: 1.3717152318387552e-13
```

Once again, the p-value falls well below conventional significance thresholds, indicating that the distribution of truth labels varies significantly across topics. The observed differences are therefore unlikely to be due to random variation within the sample.

Having examined platform- and topic-level patterns in truthfulness, we now turn to the analysis of the cognitive bias variable.

```python
np.unique(df.cognitive_bias)
```

```python
array(['appeal_to-emotion', 'appeal_to_emotion', 'appeal_to_fear',
       'confirmation_bias', 'hasty_generalization', 'none'], dtype=object)
```

The LLM exhibited some inconsistencies in labeling the *appeal_to_emotion* category, occasionally producing slight variations in spelling or closely related variants (e.g., *appeal_to_fear*).

To ensure consistency and avoid artificial fragmentation of categories, these related labels are consolidated into a single class (*appeal_to_emotion*) during the data cleaning phase.

```python
# Standardize related emotion-based bias labels into a single category
df['cognitive_bias'] = df['cognitive_bias'].apply(
    lambda x: 'appeal_to_emotion'
    if x in {'appeal_to_emotion', 'appeal_to_fear', 'appeal_to-emotion'}
    else x
)
```

After standardising the bias labels, we can examine their distribution through a dedicated visualisation.

```python
# Distribution of cognitive bias categories among posts labeled as FALSE
sns.countplot(
    data=df[df['truth_label'] == 'FALSE'],
    x='cognitive_bias',
    hue='cognitive_bias',
    stat='percent'
)

# Improve readability of category labels
plt.xticks(rotation=90)

plt.show()
```

![image.png](Part%202%20Exploratory%20analysis/image%202.png)

The distribution of cognitive bias categories is clearly imbalanced, with certain biases appearing substantially more frequently than others.

Overall, the exploratory analysis highlights several non-random structural patterns in the dataset.

First, the distribution of truth labels differs significantly across platforms, suggesting that the platform environment may be associated with distinct informational dynamics. Second, truthfulness also varies across topics, with certain themes—such as anti-vaccination discourse—showing a higher proportion of content classified as false.

It is important to emphasise that all labels are generated through automated LLM-based annotation and should therefore be interpreted as weak supervision rather than ground truth. Nevertheless, the observed statistical associations provide meaningful insights into how misinformation and cognitive biases may be distributed across platforms and topics within this sample.
