# **Cognitive Bias and Misinformation Analysis on Social Media**

This project investigates how **cognitive biases** and perceived truthfulness are distributed across social media platforms and controversial topics.

Posts are collected from **BlueSky** and **Mastodon** using their official APIs and automatically labeled using a local large language model (*phi-3:mini* via Ollama). The project then performs an exploratory data analysis (EDA) to examine structural patterns in the dataset.

The emphasis is placed on:

- distributional patterns
- statistical associations
- platform-level differences
- topic-level differences

All labels should be interpreted as **weak supervision**, not ground truth.
